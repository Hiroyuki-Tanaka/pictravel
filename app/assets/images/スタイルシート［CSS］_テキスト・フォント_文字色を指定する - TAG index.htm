文字色を指定する

color: ***;

広告
ブラウザ
プロパティ
  • color

color は、文字色を指定するプロパティです。


strong {
color: #ff0000;
}
プロパティ名 値 説明
color 色 色のコードまたはネーム

色の指定については、CSSの色指定をご覧ください。

  • CSSの色指定

参考

文書全体の文字色を指定する場合は、body要素に対してこのスタイルを指定します。

body {
background-color: #ffffff;
color: #000000;
}

文字色を指定する場合は、同時に背景色も指定しておくようにします（文字色だけを指定すると、ユーザーの環境によっては文字が見づらくなってしまうため）。指定方法の詳細は関連ページをご覧ください。
